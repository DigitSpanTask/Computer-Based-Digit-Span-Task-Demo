# Computer-Based Digit Span Task (DSB Demo)

This repository contains a lightweight demo version of the **Computer-Based Digit Span Task**, designed as a ready-to-use software tool for experimental and behavioral research.

The task is implemented as a simple browser-based application and can be run without additional setup. It is intended to demonstrate the interface, task logic, and core functionality of the software described in the accompanying manuscript.

## Features

- Digit Span Forward or Backward recall
- Configurable task parameters
- Practice round with feedback
- Multiple scored rounds with increasing difficulty
- Optional text-to-speech (TTS) support
- Works in standard modern browsers
- Suitable for lab, online, or mobile-based administration

## Files

- `index.html` – main demo page
- `styles_general.css` – styling and layout
- `scripts_digits_task.js` – task logic and behavior

## Running the Demo

No installation is required.

Simply open the hosted demo in your browser:

**Live demo:**  
https://digitspantask.github.io/DSB_Demo/

Alternatively, the task can also be run locally by opening `index.html` in a browser.

## Notes

This repository provides a **demonstration version** of the task only.  
The full experimental implementation, including integration into an **oTree** workflow and full methodological details, is described in the accompanying paper.
